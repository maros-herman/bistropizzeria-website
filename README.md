# Bistroteca & Pizzeria – Modern Restaurant Presentation

Elegant, fast and fully responsive web concept designed for a modern Italian bistro and pizzeria.

---

## About the project

This project serves as a modern digital business card for a gastro business, where the key to success is speed, clarity and a flawless experience on mobile devices.

The goal was to transform classic static menus into an interactive form that will make it easier for the customer to choose a dish and immediately lead them to book a table. The project is programmed from the ground up with an emphasis on clean code, modern UX rules in gastronomy and high performance.

## Main functions and features

* **Instant category filter:** Dynamic menu filtering (pizza, pasta, drinks) using JavaScript without annoying page reloads.
* **Online reservation form:** Intuitive and clear interface for table reservations adapted for quick filling on smartphones.
* **Mobile-First approach:** Menu and important contact information are optimized for smooth one-handed operation on mobile.
* **Multilingual version:** Full structured preparation for switching language mutations (SK/EN) for international guests.
* **High loading speed:** Optimized assets and clean semantics guarantee lightning-fast access to the daily menu even with weaker mobile connections.

## Technologies used

* **HTML5** – semantic section structure and accessibility
* **CSS3** – modern dark layout, responsiveness (Flexbox/Grid), clean typography
* **JavaScript (Vanilla)** – client logic for product filtering and form handling
